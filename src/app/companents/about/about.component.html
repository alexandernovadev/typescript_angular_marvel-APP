Información

Con esta aplicación podras elejir entre todos los heroes de Marvel y guardar en una lista de Favoritos que se almacenara en el Navegador, tambien podra Actualizar tu nombre y sera guardado el navegador, Asi recargues la pagina se almacenaran los datos.

Tecnologías

  • Angular

    Esta Aplicacion esta hecha Con Angular 8, se modularizo con MVC y se utlizo los componentes y la rutas de Angular para crear una aplicacion SPA
  • Boostrap

    Se utilizo el Framework Boostrap Para tener una desarrollo Agil y eficiente a la hora de perzonalizar el diseño de la Aplicaion
  • API rest

    Se utlizo el protocolo HTTP, para hacer consultas a la API de Marvel que nos brinda la posibilidad de tener Información de todos los personajes se su colección.
  • Tecnologias De Desarrollo

    Las tecnologias estandar para crear la web se utlizaron para la maquetación del sitio
